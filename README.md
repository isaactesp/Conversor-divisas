# Conversor de divisas con GUI
Podéis encontrar el proyecto dividido en dos capas junto a un modelo de dominio. Separamos la capa de presentación y la capa de negocio con el objetivo de hacer un proyecto extensible. Cualquiera en el futuro podrá añadir las funcionalidades que vea necesario. 
En el modelo de dominio se incluye la clase Divisa y la clase ColeccDivisa, que hace referencia a una colección de divisas. Implementada utilizando un Dictionary pero se podría utilizar KeyedCollection o cualquier otra a gusto del usuario. Las capas se han progrmado de manera que esto no afecte al usuario. 

# Despliegue
Clonar el repositorio en local y abrir el archivo "PresentacionWindows" desde Visual Studio.
